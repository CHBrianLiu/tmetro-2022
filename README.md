## Development

### Get started

1. Install dependencies.

    ```shell
    npm install -D
    ```

2. Start the development server.

    ```shell
    npm run dev
    ```

3. Open *http://localhost:3000* in your browser.

### Auto-formatting

```shell
npm run format
```

### Code quality

```shell
npm run lint
npm run check-format
```
